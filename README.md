# Tide (潮汐) 🌊

> “日程如潮水般有规律地起落，顺应时间的流淌。”

Tide (潮汐) 是一款极简、优雅、隐私优先的跨平台本地日程规划与提醒工具。我们剥离了繁杂的社交与沉余功能，让你能在一个纯净、专注的环境中，掌控自己每一天的潮起潮落。

---

## ✨ 核心功能与使用指南

### 1. 📅 智能日程编排
在主界面点击右上角的 `+` 即可新建计划，Tide 提供了极度灵活的潮汐周期（循环策略）：
* **单次/特定日期**：可精挑细选多达 10 个特定的日期，适合不规律的特殊事件（如就诊、纪念日）。
* **每日循环**：适合日常习惯养成（如早起喝水、阅读）。
* **每周特定日**：按星期精准投递（如每周一、三、五健身）。

### 2. ⏰ 毫秒级防丢失通知引擎 (Android 特化)
针对国内极度严苛的安卓定制系统（如 Vivo、小米等），Tide 在底层接入了最新的 `USE_EXACT_ALARM` 精确闹钟权限，并配备了严格的自动时区探测雷达，确保每一次倒计时都能精准拉响警报。
* **自定义提前量**：支持设置从提前 1 分钟到提前 24 小时的个性化提醒。

### 3. 🔄 绝对私有的 GitHub 云同步
无需注册账号，不依赖任何第三方中心化服务器。Tide 将数据的绝对控制权交还给你：
* **配置方法**：点击主界面左上角的头像，填入你个人的 `GitHub 私有仓库链接` 和 `Personal Access Token (需 repo 权限)`。
* **智能比对**：点击顶部的“云同步”图标，Tide 会自动比对本地与云端数据。当发生冲突时，会弹出视觉化面板供你选择“保留本地”或“拉取云端”。

### 4. 🎨 沉浸式个性化与防溢出排版
* **壁纸切换**：点击顶部的 `壁纸` 图标，支持一键切换本地相册图片或填入网络高清图源。
* **极简模式**：点击顶部的 `日历/列表` 按钮，一键在“今日待办”与“全部跨期计划”间无缝切换。

## ⚠️ 跨平台兼容性与已知 Bug 披露

Tide 致力于全平台部署，但受限于底层操作系统的系统级差异，当前各平台战况如下：

### 📱 Android (主战场 - 完美运行，需注意权限)
* **状态**：✅ 完美支持（包含瞬间通知与定时预警）。
* **已知问题 (定制 ROM 封杀)**：在 Vivo (OriginOS)、小米 (HyperOS/MIUI)、华为 (HarmonyOS) 上，系统可能会强杀后台闹钟。
* **解决方案**：必须在手机设置中手动为 Tide 开启：**① 允许自启动**；**② 允许后台高耗电（无限制）**；**③ 在多任务界面下拉加锁**。

### 🐧 Linux (次战场 - 基础可用，定时受限)
* **状态**：⚠️ 基础功能与数据同步完美运行，但**无法进行定时通知**。
* **已知 Bug**：Linux 桌面环境缺乏统一的底层定时唤醒机制。`flutter_local_notifications` 插件在 Linux 平台**未实现** `zonedSchedule` 方法。尝试触发定时通知会被底层静默拦截，但点击链路测试的瞬间弹窗可以正常工作。

### 🪟 Windows (未知领域 - 实验性平台)
* **状态**：❓ **未经充分测试**。
* **已知 Bug**：当前使用的本地通知核心引擎官方暂未完全适配 Windows 原生弹窗。在 Windows 平台编译运行可能面临报错或无法弹出系统级通知的情况。建议目前仅在 Windows 上作为无提醒的“纯粹日程看板”使用。
